Подробное описание

Основной класс. Реализует работу с кольцами многочленов над кольцом вычетов из q элементов

FieldVec(void) Конструктор. При таком создании полином представляется в виде 0 * x ^ 0

FieldVec(std::vector<unsigned int> v) Конструктор. Если в векторе v нет элементов, то полином представляется в виде 0 * x ^ 0, иначе - коэффициентом при x^i будет v[i]

Аргументы
